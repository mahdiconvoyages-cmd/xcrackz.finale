# Guide Rapide - Logos Xcrackz

## Ce dont vous avez besoin :

### Fichier 1 : logo.png
- 512x512 pixels minimum
- Fond blanc ou transparent
- Logo complet

### Fichier 2 : logo_foreground.png  
- 512x512 pixels
- Fond TRANSPARENT obligatoire
- Logo avec 20% de marge

## Placez ces 2 fichiers ici :
`assets/icons/logo.png`
`assets/icons/logo_foreground.png`

## Puis dites-moi : "génère les icônes"
